####1. 光标移动
```
hjkl    左下上右（方向键也可以）     
ctrl-f  向上翻页
ctrl-b  向下翻页
ctrl-u  向上翻半屏
ctrl-d  向下翻半屏
w       光标向后移动一个词
b       光标向前移动一个词
0       光标跳至行首
^       光标跳至第一个字符
$       光标跳至行尾
gg      光标跳至文件开始
G       光标跳至文件最后
nG      光标跳至第n行
)       移动光标到下一个句子
(       移动光标到上一个句子
*       读取光标处的字符串，并且移动到再次出现该字符串的地方
#       和*类似，只不过查找方向不一样
fx      移动光标到当前行的下一个x处，可以用分号来重复
Fx      和fx类似，也只是方向不同而已
H       移动光标到屏幕上面
M       移动光标到屏幕中间
L       移动光标到屏幕下面
zt      光标所在行到屏幕顶端
zz      光标所在行到屏幕中间
zb      光标所在行到屏幕底端
```
####2. 插入模式
```
i       在当前字符的左边插入
I       在当前行行首插入
a       在当前字符的右边插入
A       在当前行尾插入
o       在当前行下面插入一个新行
O       在当前行上面插入一个新行
c$      删除从光标位置到行尾的字符并进入插入模式
d$      删除从光标位置到行尾的字符不进入插入模式
esc     退出插入模式
ctrl-n,ctrl-p 你敲入几个字符后可以用来补全
```
####3. 编辑模式
```
r       替代单个字符
R       替代字符串
>>      向右缩进
<<      向左缩进
u       undo刚才的操作
ctrl-r  redo刚才的操作
cc      删除当前行并进入插入模式
cw      删除当前word并进入插入模式
s       删除光标所在字符并进入插入模式
S       删除当前行并进入插入模式
.       重复上一个命令
```
####4. 剪切粘贴
```
x       删除(cut)当前字符
X       删除(cut)前一个字符
dd      删除当前行
ndd     删除n行
dw      删除当前word
D       删除光标所在字符到行尾
yy      复制一行
2yy     复制两行 
Y       复制一行
yw      复制当前word
y$      复制当前字符到行尾
p       粘贴到光标所在行的下一行
P       粘贴到光标所在行的上一行
```
####5. 查找替换
```
/text   向后查找text
?text   向前查找text
n       向同一个方向查找
N       向反方向查找
:s/old/new/g  用new替换所有的old
:s/old/new/gc 用new替换所有的old并确认
```
####6. 多window
```
:sp file  横向再打开一个文件
:vsp file 纵向再打开一个文件
ctrl-w w  在不同window中切换
ctrl-w q  退出一个window
ctrl-w s  再水平分一个window
ctrl-w v  再纵向分一个window
```
####7. 其他
```
:ab email benbenbear@example.com   下次在编辑时输入email就会自动转换成benbenbear@example.com,如果想暂时取消可以在输入后按ctrl-v，或者也可以通过:una来取消
:4,9d    删除4到9行
:3,9t.   把3到9行拷贝到当前行
:t3.     把当前行拷贝到第3行 
/test\c  可在搜索时忽略大小写
```
####`vim.rc`最常用配置
```
syn on                语法支持
set ai                自动缩进
set nu                显示行号
set showmatch         代码匹配
set ignorecase        检索时忽略大小写
set foldmethod=syntax 代码折叠
```
